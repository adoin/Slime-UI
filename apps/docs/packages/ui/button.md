# Button 按钮

Button 按钮基础按钮组件。基础用法基础的按钮用法。

<!-- <script setup>
import { VButton } from '@mylib/ui'
</script>

<VButton>默认按钮</VButton> -->

| 属性     | 参数     | 说明                             | 类型      | 默认值 |
| -------- | -------- | -------------------------------- | --------- | ------ |
| type     | 按钮类型 | 'primary' \| 'default' \| 'text' | 'default' |
| size     | 按钮大小 | 'large' \| 'default' \| 'small'  | 'default' |
| disabled | 是否禁用 | boolean                          | false     |
